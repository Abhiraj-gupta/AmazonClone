**Amazon Clone**

A full-stack e-commerce application inspired by Amazon, built to demonstrate modern web development practices including REST APIs, authentication, database relationships, and full-stack deployment.

The application allows users to browse products, manage a cart, maintain a wishlist, and authenticate securely.
